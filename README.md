## Description ##

Provides UI for connecting from one WordPress installation to another via the WP REST API over <a href="https://github.com/WP-API/OAuth1">OAuth1</a>

## Installation ##

### Manual Installation ###

1. Upload the entire `/wds-rest-connect-ui` directory to the `/wp-content/plugins/` directory.
2. Activate WDS WP REST API Connect UI through the 'Plugins' menu in WordPress.
